========================================
   ISUX 运营视觉加油包 v1.1 - 使用说明
========================================

【快速启动】

  macOS 用户：
  双击「start.command」即可自动启动并打开浏览器

  首次运行可能需要授权：
  1. 右键点击「start.command」
  2. 选择「打开」
  3. 在弹窗中点击「打开」确认

----------------------------------------

【手动启动（备选方案）】

  如果双击无效，可手动操作：

  1. 打开「终端」应用（在 启动台 > 其他 中）
  2. 输入以下命令（注意替换路径）：

     cd 你解压的文件夹路径
     python3 -m http.server 8000

  3. 打开浏览器访问：
     http://localhost:8000/index.html

----------------------------------------

【常见问题】

  Q: 为什么不能直接双击 HTML 文件打开？
  A: 浏览器安全限制，本地文件无法加载 JS 模块，
     必须通过 HTTP 服务器访问。

  Q: 端口被占用怎么办？
  A: 启动脚本会自动寻找可用端口（8000-9000）

  Q: 如何关闭服务器？
  A: 关闭终端窗口即可

----------------------------------------

【文件说明】

  index.html  - 主工具页面
  start.command - 一键启动脚本
  assets/                  - 资源文件（请勿删除）
  favicon.png / icon.png   - 图标文件
